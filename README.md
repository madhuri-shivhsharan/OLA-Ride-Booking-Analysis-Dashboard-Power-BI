# OLA-Ride-Booking-Analysis-Dashboard-Power-BI
Developed an interactive Power BI dashboard to analyze OLA ride booking data and generate business insights from ride operations. The project focuses on monitoring booking performance, revenue trends, cancellation patterns, vehicle usage, and customer behavior using data visualization and KPI reporting techniques.
<img src="Ola Project Overview Page.png" width="800">
# Objective
- Created an interactive Power BI dashboard to analyse OLA ride booking data and customer trends. 
- Used page navigation, slicers, KPI cards, and charts to improve dashboard interaction and reporting. 
- Analysed bookings, revenue, cancellations, vehicle types, and pickup locations to generate business insights. 
- Used Power BI, DAX, and Power Query for data cleaning, visualization, and dashboard development.

# The dashboard includes:
- Total Bookings Analysis
- Revenue Tracking
- Customer & Driver Cancellation Analysis
- Booking Status Breakdown
- Vehicle Type Performance
- Top Pickup Locations
- Interactive Filters and Page Navigation
# Tools & Technologies Used
- Power BI
- SQL
- Microsoft Excel / CSV Dataset
# Data Used
The dataset contains OLA ride booking information including:
- Booking ID
- Date & Time
- Vehicle Type
- Pickup & Drop Locations
- Booking Status
- Payment Method
- Ride Distance
- Revenue
- Customer Cancellation
- Driver Cancellation
# Key Features
- KPI Cards for Total Bookings & Revenue
- Booking Trend Analysis
- Cancellation Analysis
- Vehicle Type Breakdown
- Top Pickup Locations Analysis
- Interactive Filters & Navigation
